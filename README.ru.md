## Обзор

Fetcho — лёгкое расширение для браузера, написанное на TypeScript и собранное с помощью Vite. Оно обнаруживает ссылки на видео в буфере обмена, определяет платформу (Instagram, Twitter/X или YouTube Shorts) и автоматически перенаправляет их на соответствующий сервис загрузки. Обратный отсчёт в 3 секунды даёт возможность отменить действие до начала процесса.

## Возможности

- **Автоопределение**: Считывает буфер обмена при открытии попапа и определяет поддерживаемые платформы
- **Мультиплатформенность**: Instagram, Twitter/X и YouTube Shorts
- **Обратный отсчёт 3 секунды**: Автоматический запуск загрузки с возможностью отмены
- **Ручная вставка**: Вставьте ссылку напрямую, если автоопределение было пропущено
- **Автоматизация контент-скриптов**: Автозаполнение полей ввода и нажатие кнопок загрузки на сайтах-загрузчиках
- **Чистый UI попапа**: Минималистичный и доступный интерфейс с бейджами платформ

## Стек технологий

## Начало работы

### Требования

- Node.js 18 или выше
- Браузер на базе Chromium (Chrome, Edge, Brave и т.д.)

### Установка

```bash
# Установка зависимостей
npm install

# Сборка расширения
npm run build
```

Собранное расширение будет помещено в директорию `dist/`.

### Загрузка в браузер

1. Откройте браузер и перейдите на `chrome://extensions/`
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **Загрузить распакованное**
4. Выберите папку `dist/` из этого проекта
5. Иконка расширения появится на панели инструментов

## Использование

1. Скопируйте ссылку на видео из Instagram, Twitter/X или YouTube Shorts
2. Нажмите на иконку расширения на панели инструментов
3. Попап автоматически определит платформу и запустит обратный отсчёт в 3 секунды
4. Нажмите **Отмена**, чтобы остановить автозапуск, или дождитесь его завершения
5. Откроется новая вкладка с сайтом-загрузчиком, где ссылка будет уже вставлена и загрузка начата

Вы также можете вставить ссылку вручную в поле ввода и нажать **Загрузить**.

## Как это работает

1. **Определение из буфера**: При открытии попапа расширение считывает буфер обмена и проверяет, соответствует ли ссылка поддерживаемой платформе
2. **Маршрутизация платформ**: Определённая платформа определяет, какой сервис загрузки использовать:
   - Instagram -> `igram.world`
   - Twitter/X -> `ssstwitter.com`
   - YouTube Shorts -> `ytshortsdl.io`
3. **Передача через хранилище**: Ссылка и идентификатор платформы сохраняются в `chrome.storage.local`
4. **Автоматизация контент-скриптом**: Контент-скрипт на сайте-загрузчике считывает сохранённую ссылку, заполняет поле ввода и нажимает кнопку загрузки

## Разрешения

| Разрешение | Причина |
|---|---|
| `clipboardRead` | Автоопределение ссылок на видео из буфера обмена |
| `storage` | Передача данных ссылки и платформы между попапом и контент-скриптами |
| `tabs` | Открытие сайтов-загрузчиков в новых вкладках |

> [!IMPORTANT]
> Это расширение перенаправляет ссылки через сторонние сервисы загрузки. Оно не хранит и не обрабатывает видеоконтент самостоятельно. Используйте ответственно и уважайте права создателей контента.

## Разработка

```bash
# Запуск dev-сервера с горячей перезагрузкой модулей
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр собранной версии
npm run preview
```

Во время разработки используйте `npm run dev` и перезагружайте расширение из `chrome://extensions/` после изменений. Плагин CRXJS поддерживает HMR для попапа и контент-скриптов.
